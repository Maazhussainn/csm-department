# Privilege Escalation Attack Detection and Mitigation in Cloud using Machine Learning

## Description
This project is a machine learning-based system designed to detect and mitigate privilege escalation attacks in cloud environments using models like CatBoost, XGBoost, and LightGBM.

## Project Structure
- Main.py → Main execution file  
- test.py → Testing script  
- Dataset/ → Dataset used  
- docs/ → Project report, abstract, and screenshots  

## Requirements
Install dependencies using:
pip install -r requirements.txt

## Note
Recommended Python version: 3.7/3.8
